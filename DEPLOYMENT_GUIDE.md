## 📱 배포 후 테스트 시나리오

### 시나리오 1: 신청자 회원가입 → 대리입찰 신청
1. http://bid-tok.kr 접속
2. 우측 상단 "로그인" 클릭
3. "회원가입" 탭에서 신청자 회원가입
   - 아이디 중복 확인
   - 이메일 중복 확인
   - 휴대폰 인증 (6자리 코드)
   - 비밀번호 8자 이상
   - 필수 약관 동의
4. 로그인 후 메인 페이지에서 "신청하기" 클릭
5. 대리입찰 신청서 작성
   - 사건번호 조회
   - 입찰가 입력 (최소 10만원)
   - 서명 입력
6. 마이페이지에서 신청 내역 확인

### 시나리오 2: 법무사 회원가입 → 배정 수락
1. 메인 페이지에서 "전문가 신청" 버튼 클릭
2. 자동으로 회원가입 페이지 이동 (법무사 탭)
3. 법무사 회원가입
   - 법무사 등록번호 입력
   - 서비스 지역 선택 (17개 광역시도)
   - 사무소 주소 입력
   - 증명서 업로드
4. 관리자 승인 대기
5. 승인 후 로그인 → 법무사 마이페이지
6. 1시간 타이머 배정 요청 확인
7. 수락/거부 처리

### 시나리오 3: 관리자 로그인
1. http://bid-tok.kr/admin.html 직접 접속
2. 관리자 계정 로그인:
   - ID: bootv1 / PW: 1234 (마스터)
   - ID: bootv2 / PW: 1234 (팀장)
   - ID: bootv3 / PW: 1234 (팀장)
   - ID: bootv4 / PW: 1234 (팀장)
3. 대시보드 통계 확인
4. 법무사 승인 처리
5. 신청 내역 관리
6. 정산 처리

## ⚠️ 알려진 이슈 및 제한사항

### 현재 임시 동작 (API 연동 전)
1. **로그인**: 임시 계정으로 로그인 (실제 DB 검증 없음)
2. **SMS 인증**: 콘솔에 인증번호 출력 (실제 SMS 발송 안 됨)
3. **중복 확인**: 랜덤 중복 판정 (실제 DB 조회 안 됨)
4. **결제**: 결제 버튼만 존재 (Toss API 미연동)
5. **파일 업로드**: 파일 선택만 가능 (실제 업로드 안 됨)
6. **데이터 조회**: LocalStorage 기반 (실제 DB 연동 안 됨)

### 모바일 대응
- 모든 페이지 반응형 디자인 적용
- 최소 해상도: 320px (iPhone SE)
