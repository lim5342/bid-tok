# 대리입찰 톡 (Proxy Bidding Talk)

부동산 경매 입찰의뢰인과 매수신청대리인·법무사를 연결하는 중개 플랫폼입니다.

---

## 🎯 프로젝트 개요

**대리입찰 톡**은 부동산 경매 입찰 시 거리·시간 제약으로 어려움을 겪는 분들을 위해, 전국의 전문가(매수신청대리인·법무사)와 간편하게 매칭해주는 서비스입니다.

### 주요 특징
- **3분 완성** – 사건 조회 → 정보 입력 → 전자계약 → 결제
- **전국 커버** – 서울부터 제주까지 16개 시·도 전문가 매칭
- **투명한 수수료** – 매수신청대리인 70,000원 / 법무사 100,000원 (VAT 포함)
- **안전 보장** – 보증금 최대 5억 원 배상책임보험
- **즉시 정산** – 베타 기간 입찰 완료 익일 즉시 입금 (전문가 대상)

## 🚀 주요 기능

### 1️⃣ **일반 사용자 (입찰 신청자)**

#### **STEP 1: 사건 조회**
- 16개 시·도, 70+ 법원 드롭다운 선택
- 사건번호·물건번호 입력
- 대법원 경매 사이트 직접 링크 (https://www.courtauction.go.kr)
- 조회 결과: 사건정보, 감정가, 최저입찰가, 입찰보증금 표시

#### **대리인 유형 선택**
- **💰 매수신청대리인** (70,000원)
  - 저렴한 수수료
  - 주민센터 방문 필요 (인감증명서)
  - 서류 우편 제출
  - 2~3일 소요
- **⚖️ 법무사** (100,000원) **[추천]**
  - 온라인 간편 처리
  - 주민센터·인감 불필요
  - 전자서명만 (3분)
  - 10분 완료

#### **STEP 2: 입찰 정보 작성**
- 입찰가 입력 (콤마 자동 포맷)
- 보증금 자동 계산 (10%)
- 신청자 정보 (이름, 계좌, 주민번호 마스킹)
- 주소 검색 (Daum 우편번호 API)
- 휴대폰 인증 버튼
- 약관 동의 체크
- 자동 저장 (LocalStorage)

#### **STEP 3: 전자 계약**
- 위임계약서 온라인 표시
- HTML5 Canvas 서명
  - PC: 마우스 드래그
  - 모바일: 터치 드로잉
- 서명 완료 → PNG 이미지 저장
- 계약 내용 자동 입력 (당사자, 사건, 수수료)

#### **STEP 4: 수수료 결제**
- 선택한 대리인 유형에 따라 금액 표시 (70,000원 / 100,000원)
- 서비스 포함 내역
- 유의사항·환불 규정
- 포인트 사용 (준비 중)
- **결제 완료 → 신청 데이터 DB 저장**

#### **뒤로 가기**
- STEP 2 ↔ STEP 3 자유 이동
- 입력 데이터 유지 (자동 저장/로드)

---

### 2️⃣ **전문가 (매수신청대리인·법무사)**

#### **전문가 신청 페이지**
- 상단 네비게이션 "전문가 신청" 버튼 (골드 색상)
- **기본 정보**
  - 전문가 구분 (라디오): 매수신청대리인 / 법무사
  - 성명·자격증 번호·연락처·이메일
- **사무실 주소** (Daum 우편번호 API)
- **활동 지역** (전국 16개 시·도 다중 선택)
- **서류 업로드** (자격증·보험증 파일)
- **희망 수수료** 입력
- **카카오톡 알림 수신 번호**
- **소개글** (Textarea)
- **약관 동의** 체크
- **신청 완료 → DB 저장 (experts 테이블)**

#### **전문가 대시보드** (향후 개발 예정)
- 고유 링크로 접속 (예: `/expert/EXP001`)
- 배정된 입찰 건 조회
- 입찰 결과 입력 (낙찰/유찰)
- 정산 내역 확인

---

### 3️⃣ **관리자 (Admin Dashboard)**

#### **접근 방법**
- 로고 5번 연속 클릭 (비밀 접근)

#### **대리입찰 신청 관리 탭**
- **실시간 신청 목록**
  - 신청일시, 신청자, 연락처, 사건번호, 법원, 입찰가, 상태
- **상세보기 버튼** → 팝업으로 전체 정보 표시
  - 신청 정보, 경매 정보, 입찰 정보, 전문가 정보, 처리 상태
- **엑셀 다운로드** 버튼
  - 전체 신청 내역 CSV 파일 생성
  - UTF-8 BOM 포함 (한글 깨짐 방지)
- **상태 관리** (접수·매칭중·입찰완료·낙찰·유찰·취소)

#### **전문가 관리 탭**
- **승인 대기 목록**
  - 신청일시, 성명, 구분, 자격증번호, 연락처, 활동지역
- **승인/거절 버튼**
  - 승인 → 상태 "승인"으로 변경, 승인일시 기록
  - 거절 → 거절 사유 입력 후 상태 "거절"로 변경
- **엑셀 다운로드** 버튼
  - 전문가 목록 CSV 파일

## ⚖️ 법적 고지

본 플랫폼은 입찰의뢰인과 매수신청대리인·법무사를 연결하는 **중개 플랫폼**입니다.

- 중개 수수료는 **무료**이며, 입찰 수수료는 전문가에게 직접 지불됩니다.
- 입찰 대리 업무는 배정된 전문가가 독립적으로 수행하며, 플랫폼은 중개 서비스만 제공합니다.
- 입찰 결과(낙찰·유찰)에 대한 책임은 입찰의뢰인과 전문가 간 계약에 따릅니다.
- 플랫폼은 전문가의 자격·보험 가입 여부를 확인하지만, 입찰 과정에서 발생하는 문제에 대해 직접적인 책임을 지지 않습니다.
- 본 서비스 이용 시 [이용약관] 및 [개인정보처리방침]에 동의한 것으로 간주됩니다.

✅ **4단계 입찰 신청** – 사건 조회, 대리인 선택, 정보 입력, 전자서명, 결제  
✅ **전문가 신청** – 기본 정보, 주소 검색, 지역 선택, 서류 업로드, DB 저장  
✅ **관리자 대시보드** – 신청 관리, 전문가 승인/거절, 엑셀 다운로드, 상세보기

✅ **반응형 디자인** – Desktop / Tablet / Mobile 최적화  
✅ **자동 저장** – LocalStorage 활용, 뒤로가기 데이터 유지  
✅ **전자서명** – HTML5 Canvas (PC 마우스, 모바일 터치)
